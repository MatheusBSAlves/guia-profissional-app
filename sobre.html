Sobre o projeto

Projeto de extensão acadêmico voltado à preparação da comunidade para processos seletivos.

Justificativa

A primeira barreira para o emprego é, muitas vezes, a incapacidade de estruturar um currículo profissional ou de se portar adequadamente em uma entrevista. Esta plataforma elimina a necessidade de softwares complexos e oferece um caminho gratuito, simples e acessível para a comunidade.

Objetivo geral

Desenvolver uma plataforma web interativa de auxílio à carreira, integrando um simulador de perfil de entrevista com um gerador dinâmico de currículos.

Tecnologias

  • HTML5 semântico
  • CSS3 puro em arquivos externos
  • JavaScript Vanilla moderno em arquivos externos
  • Sem frameworks, sem bibliotecas externas, sem backend, sem banco de dados

Acessibilidade e responsividade

Uso de tags semânticas, hierarquia de títulos, foco visível, contraste adequado, navegação por teclado, atributos ARIA quando necessário e layout responsivo via media queries.
